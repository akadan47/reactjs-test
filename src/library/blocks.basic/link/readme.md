## Блок ссылки (link)

#### Модификаторы:

* Состояния (`block`)
* С подчеркиванием ('underline')
* Без подчеркивания (`underline_none`)
* Успех (`success`)
* Ошибка (`error`)
* Белый цвет (`white`)
* Черный цвет (`black`)
* Серый цвет (`gray`)
* Предупреждение (`warning`)


#### Пример PUG вызов
```html
+link('link_default')
```
